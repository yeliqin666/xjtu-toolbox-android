# 更新日志

## <span class="changelog-version">3.4.1</span>

- 🔑 **【考勤】** 修复登录 URL 协议错误（http → https），解决考勤无法登录的问题
- ⬆️ **【更新】** 应用内下载始终可用：Gitee Release 附件缺失时自动 fallback 到直链，移除跳转浏览器按钮
- 🗓️ **【课表】** 修复浮动胶囊 dock 下三个 Tab 底部出现固定空白的问题，现在与其他页面一致自然延伸

## <span class="changelog-version">3.4.0</span>

- 🗓️ **【课表】** 时间轴空时段自动纵向压缩，切周时先恢复均匀再平滑压缩，左侧时间标签同步收缩
- 🗓️ **【课表】** 学期总览也启用空时段压缩，长时间无课的小时区段不再占满高度
- 🏠 **【首页】** 「全部服务」改为真正的瀑布流布局，卡片高低错落更紧凑美观
- 🏠 **【首页】** 快捷入口与服务卡提示根据使用习惯动态变化，常用服务自动置顶
- ✏️ **【自建日程】** 添加日程弹窗全面 MIUIX 化：精简 NumberPicker、移除备注输入框避免键盘遮挡，结束时间随开始自动调整
- 🎫 **【加餐券】** 移除右上角刷新按钮，改用 MIUIX 下拉刷新，空状态/错误状态也可下拉重试
- ⚙️ **【设置】** 修复弹窗在某些情况下不显示的问题，清除缓存后大小立即刷新
- 📝 **【更新日志】** 统一三处来源：启动弹窗、设置页与云端 Release Notes 全部从同一份数据派生，并支持跨版本堆叠展示
