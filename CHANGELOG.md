# Changelog

本文件记录应用源码层面的重要变更。

## v3.5.0 - 2026-05-15

### 功能新增与优化
- **MFA 两步验证弹窗**：`autoLogin` 遇到两步验证需求时，自动弹出验证码对话框让用户即时完成验证并继续使用功能，不再静默失败。
- **校外 WebVPN 扩展**：教务系统（JWXT/JWAPP）、思源学堂（LMS）、场馆预订（Venue）、教材中心（Jiaocai）在校外自动启用 WebVPN 通道。
- **教材中心精简**：移除在线阅览和 PDF 下载功能，保留教材搜索与书目信息展示。
- **miuix 升级**：UI 组件库 miuix 0.9.0 → 0.9.1（SpinnerEntry → DropdownItem 迁移）。

### Bug 修复
- **校园卡付款码误标收入**：修复付款码消费记录被错误标注为"收入"的问题，改为白名单模式（仅充值/圈存为收入）。
- **登录态过期提示**：各子系统 `reAuth` 失败时统一抛出 `AuthExpiredException`，UI 给出精确过期提示而非笼统的"网络不佳"。

## v3.3.0 - 2026-05-05

### 功能新增与优化
- **电子加餐券**：新增加餐券查询页面，支持可领取、可使用、已用完、已过期筛选，展示余额、有效期、使用状态与券图。
- **统一认证**：加餐券接入自动登录与共享 CAS 会话，补充电子券系统 JWT 换取、失效重试与接口错误提示。
- **首页与工具入口**：首页“全部服务”和工具页“校园服务”均新增“加餐券”入口。
- **设置页**：统一为应用二级页面风格，修复部分中文文案显示异常和 LMS 下载路径与标题重叠问题。
- **日程缓存**：新增课表优化缓存读取，改进节假日过滤逻辑与桌面小组件稳定性。

---

## v3.2.0 - 2026-05-01

### 功能新增与优化
- **空教室查询**：支持校区与教学楼选择的持久化保存与读取，优化启动体验。
- **课表与假期**：
  - 新增假期日历 API，支持动态获取法定节假日信息及缓存。
  - 课表视图新增节假日名称显示。
  - 课表导出 ICS 文件时自动跳过假期日期，并新增了课程提醒功能。
  - 过滤主时间线的合并课程，排除落在假期内的无效课程。
- **自定义课程**：增强周次解析与假期冲突检测，支持排课冲突检查逻辑。
- **工程结构**：新增自动化更新 API 的脚本 `update_api.py`。

## v3.0 - 2026-03-18

> 对比基线：`origin/main@d0d3786`（与本地 HEAD 同基线），变更来源为工作区相对远程差异（`git diff origin/main`）。

### 导航与入口

- `MainActivity` 导航重构：原“教务”Tab 调整为“课程”Tab，课程功能聚合到主导航。
- 首页多个课表入口统一跳转为 `MAIN + COURSES`，减少旧路由分叉。
- 兼容历史路由 `Routes.SCHEDULE`，统一映射到课程 Tab。
- 新增小组件唤起参数 `EXTRA_LAUNCH_TAB`，支持直接落到指定底部 Tab。

### 课程页与课表体验

- `ScheduleScreen` 支持嵌入式模式（`showTopBar=false`），用于主界面课程 Tab 场景。
- 学期区 + Tab 结构优化，视觉风格调整为更贴近 MIUIX 的轻分组样式。
- 加入 `isRefreshingFromNetwork` 状态与细进度条，缓存回显后后台刷新更可感。
- 修复“先离线渲染、后登录恢复”场景：登录可用后自动触发在线刷新。
- 教材卡片、考试卡片、筛选交互等细节样式与信息层级优化。

### 桌面小组件（稳定性重点）

- 课表/校园卡小组件统一稳定链路：`RemoteViews + AppWidgetProvider (+ RemoteViewsService)`。
- 课表小组件：
  - 增加周/日切换逻辑优化（跨周边界处理）。
  - 2×2、4×2 两规格统一绑定流程，减少重复代码。
  - 点击入口直达课程 Tab，避免旧路由链路。
- 校园卡小组件：
  - 2×2 布局重排，三餐数据显示更紧凑，减少“晚饭溢出”。
  - 金额格式更紧凑（按量级保留小数），可读性提升。
  - `appwidget-provider` 规格参数调整为更贴合 2×2 实际占位。
- 新增样式资源：`bg_widget_action_pill.xml`、`bg_widget_chip.xml`。

### 业务功能修复与增强

- 评教：
  - 自动评教兜底匹配增强（题号/答案码/题干多路径匹配），降低题库字段漂移影响。
  - 撤回评教交互与状态反馈优化。
  - 研究生评教确认弹窗底部安全区与输入法遮挡适配完善。
- 成绩：GPA/均分计算逻辑优化，均分仅统计有效数值成绩学分。
- 场馆：日期切换刷新逻辑与收藏动画触发条件优化。
- 校历：学期事件按开始日期排序，展示更稳定。

### UI 与系统行为优化

- 主题层补充系统状态栏/导航栏明暗外观同步逻辑。
- 多页面容器与卡片配色向“轻边界”方向统一，减少厚重包裹感。
- 输入法与导航栏底部内边距在部分弹窗/表单场景补齐。

### 工程与发版相关

- 应用版本升级到 `3.0`（`versionCode=14`）。
- README 版本徽标与发版 tag 示例更新到 `v3.0`。
- 工作流上传超时配置调整（详见 `.github/workflows/build.yml`）。
